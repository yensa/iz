alias WantDescriptorEvent = void delegate(
    IstNode node, 
    ref Ptr descriptor, 
    out bool stop)

Prototype of the event triggered when a serializer misses a property descriptor.

Parameters

node

The information used to determine the descriptor to return.

descriptor

What the serializer want. If set to null then node is not restored.

stop

the callee can set this value to true to stop the restoration process. According to the serialization context, this value can be noop. 

alias WantObjectEvent = void delegate(
    IstNode node, 
    ref Object obj, 
    out bool fromRefererence)

Prototype of the event called when a serializer failed to get an object to deserialize.

Parameters

node

The information the callee uses to set the parameter serializable.

obj

The Object the callee has to return.

fromReference

When set to true, the serializer tries to find the Object using the ReferenceMan.
