public struct PropDescriptor(T)

Describes a property declared in an aggregate.

A property is described by a name, a setter and a getter. Severals constructors allow to define the descriptor using a setter, a getter but also a pointer to the targeted field. Addional information includes an iz.types.RunTimeTpeInfo structure matching to the instance specialization.
