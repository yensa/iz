public class MemoryStream : Stream, StreamPersist, FilePersist8

Implements a stream of contiguous, GC-free, heap-memory. Its maximal theoretical size is 2^32 bytes (x86) or 2^64 bytes (x86_64). Its practical size limit is damped by the amount of remaining DRAM. This limit is itself reduced by the memory fragmentation.

MemoryStream is also enhanced by implementing the interfaces StreamPersist and FilePersist8. This allows to save the content either to another stream or to a file and to load the content either from another Stream or from a file.
