public class SerializableReference : PropertyPublisher

Makes a reference serializable.

The reference must be stored in the ReferenceMan. A "referenced variable" is typically something that is assigned at the run-time but not owned by the entity that want to keep track of it, or that is serialized by another entity. Note that this class is not needed to serialize a reference to an object or to a delegate, since those reference types are automatically handled by the serializer.
